This webapp is built using mvn clean package. Now added docker build to Jenkins Build Step. This is a fully automatic pipeline. Changed !.
Then run using java -jar target/jarfile.jar
Also it is set to run inside a Docker container from Amazon AWS.
These are the steps:
1) IntelliJ > create code
2) Maven build jar file
3) Create Dockerfile
4) Push all to GitHub
5) Create AWS server (Ubuntu instance) for Docker and Jenkins
6) Install and configure Jenkins on 5)
8) Create Jenkins build-job:
a) Download code from GitHub
b) Add build steps for Maven and Docker 9) Create webhook, listening for GitHub commit. On commit, this will envoke the job 8)
10) Start the first build: In Jenkins, select your project, and click "Build Now"
11) Enjoy some coffee.
